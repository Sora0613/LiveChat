## About This Project

このプロジェクトは、Laravelで非同期処理を行う練習として作成したものです。
非同期処理には、ajaxを使用しています。
実際に動かすには、Laravelの環境構築と、npmのインストールが必要です。(下記参照)

```angular2html
$ npm install && npm run dev
```
